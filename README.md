# Hotel Tycoon API REST

API RESTful utilisée pour la gestion d'un hôtel.

## Environnement de développement

### Prérequis

* ruby >= 2.2.2
* rails >= 5.0.0.rc1
* bundler < 2.0, >= 1.3.0

### Installation

```
$ git clone https://github.com/ZiiCEagle/Hotel-Tycoon-API-Rails.git
$ cd Hotel-Tycoon-API-Rails
$ bundle
$ rake db:migrate
$ rails server
```
